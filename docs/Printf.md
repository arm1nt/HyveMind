# Printf

## Overview of supported format specifiers

- `%%`

*Description:* Print a `%` character

- `%s`

*Description:* Print a string. The corresponding argument is expected to be a pointer to a **null-terminated** `char` array

- `%lu`

*Description:* The corresponding argument is expected to be of type `unsigned long` and will be printed in decimal form

- `%lx`

*Description:* The corresponding argument is expected to be of type `unsigned long` and will be printed in hexadecimal form
